Post-install

First make sure you have Eerie installed, if not, the check out the introductory page. To be able to use eerie command you have to either start a new terminal session or simply reload your bash profile: 

$ . ~/.bash_profile

Not all operating systems sotre bash profile in the same file, OS X keeps it in ~/.bash_profile, while some Linux it can be distributons in ~/.bashrc, or ~/.profile file. Make sure you reload the right one.

Now try running eerie command, if it prints nothing it means you successfully installed it! Bravo!

Concepts

Before we dive into eerie command, we should understand how Eerie actually works.

Just like Rip, virtualenv and Boid, Eerie uses the concept of multiple environments which can keep completely different packages (including different versions).
